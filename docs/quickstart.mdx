## Introduction
In this guide, we'll walk you through:

>**Installation:** How to install SuperAgentX on your system.<br />
**Create an Application:** Set up a new SuperAgentX project with the basic structure.<br />
**Virtual Environment Setup:** How to create and configure a Python virtual environment for your project.<br />
**Building a Use Case:** Customizing the pipe.py file to fit your application's needs.<br />
**Run the Application:** Finally, how to execute your new project with a simple command.

```shell
$ python3.12 -m pip install superagentx
```

<Note>
Note: <br />
After installation, the `superagentx-cli` will be available in the path `~/.local/bin` on `linux`. You may need to add
this to your environment's path variable.
</Note>

## Create an Application
To create a new `superagentx` project, run the following command in your terminal. This will prompt you to create
project with the basic structure set up for your `superagentx`.

```shell
$ superagentx-cli create
```

```shell

# Then you have to enter pipe name or it will take a default application name.
```

### Build Your Use Case

If you're building a new use case, You need to modify the file at `/home/ben/Content Creator/content_creator/pipe.py`.
The path in this file will be generated based on the input you provide through the SuperAgentX application. In that
file you can do your changes.

```python

from superagentx.agent import Agent
from superagentx.agentxpipe import AgentXPipe
from superagentx.engine import Engine
from superagentx.llm import LLMClient
from superagentx.memory import Memory
from superagentx.prompt import PromptTemplate

# Import handlers

# Example

from superagentx.handler import SerperDevToolHandler
from superagentx.handler import AIHandler
from superagentx_handlers import ScrapeHandler


async def get_content_creator_pipe() -> AgentXPipe:
    # LLM Configuration
    llm_config = {
        'llm_type': 'openai'
    }
    llm_client = LLMClient(llm_config=llm_config)

    # Enable Memory
    memory = Memory(memory_config={"llm_client": llm_client})

    # Example
    serper_handler = SerperDevToolHandler()
    crawler_handler = ScrapeHandler()
    ai_handler = AIHandler(
        llm=llm_client
    )


    # Prompt Template
    prompt_template = PromptTemplate()

    # Example - Engine(s)
    serper_engine = Engine(
        handler=serper_handler,
        llm=llm_client,
        prompt_template=prompt_template
    )
    crawler_engine = Engine(
        handler=crawler_handler,
        llm=llm_client,
        prompt_template=prompt_template
    )
    ai_engine = Engine(
        handler=ai_handler,
        llm=llm_client,
        prompt_template=prompt_template
    )

    # Create Agent with serper, crawler, ai Engines execute in Parallel - Search Products from user prompts
    serper_agent = Agent(
        name='serper Agent',
        role=f'You are the website link extractor.',
        goal='Generate the list of website urls',
        llm=llm_client,
        prompt_template=prompt_template,
        engines=[serper_engine]
    )

    crawler_agent = Agent(
        name='Crawler Agent',
        goal="Extract the content",
        role="You are the Content Extractor",
        llm=llm_client,
        prompt_template=prompt_template,
        engines=[crawler_engine]
    )

    ai_agent = Agent(
        name='AI Agent',
        goal="Summarize the given content",
        role="You are the summarization.",
        llm=llm_client,
        prompt_template=prompt_template,
        engines=[ai_engine]
    )


    # Create Pipe - Interface

    # Pipe Interface to send it to public accessible interface (Cli Console / WebSocket / Restful API)
    pipe = AgentXPipe(
        agents=[serper_agent, crawler_agent, ai_agent],
        memory=memory
    )
    return pipe
```

### Run the Application
Once everything is set up, you can run your application with this command:

```shell
$ python3.12 Content Creator/iopipe.py
```
This will start your SuperAgentX application and execute the tasks you've configured.
